# Clarity

Приложение для управления личными задачами на Python + Tkinter с элементами геймификации.

## Запуск

```bash
python main.py
```

Требования: Python 3.x (Tkinter встроен, дополнительных установок не нужно)

## Структура проекта

```
Clarity/
├── main.py        # Основной файл: интерфейс и логика приложения
├── storage.py     # Сохранение и загрузка данных (JSON)
├── data/
│   ├── tasks.json   # Файл с задачами (создаётся автоматически)
│   └── balance.json # Баланс монет и прогресс персонажа (создаётся автоматически)
└── README.md
```

## Функционал

### Задачи
- Добавление задачи по Enter
- Удаление задачи с выбором причины: «Неактуально» или «Не выполнено»
- Отметка задачи как выполненной — задача уходит вниз списка и зачёркивается
- Перемещение задач вверх/вниз стрелками
- Автосохранение при каждом действии

### Геймификация
- **Монеты 🪙** — за выполнение задачи начисляется +1, за удаление по причине «Не выполнено» снимается −2 (минимум 0)
- **Персонаж** — 8 уровней богатства: от Бездомного до Миллионера
- **Магазин 🏪** — трать монеты на предметы (Старый телефон → Велосипед → Подержанный автомобиль → Дом → Бизнес → Особняк → Яхта), каждый повышает статус персонажа
- **Описание статуса** — клик по статусу персонажа открывает попап «Кто ты?»
